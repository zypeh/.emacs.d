Zypeh's Emacs Configs

since 2012

This is an emacs configuration tree that support a number of programming languages.

To install, clone this repo to ~/.emacs.d. Ensure that the 'init.el' contained in this repo ends up at ~/.emacs.d/init.el.

Before starting up Emacs, be sure you have install sbcl and slime or else it will pop a warning message.

Some of the tips or tricks might be useful to others, and some of them are not...... that might annoy people who aren't me.
